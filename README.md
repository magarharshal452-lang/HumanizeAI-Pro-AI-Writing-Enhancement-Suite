# AI Humanizer

A custom AI Humanizer built using:

- Streamlit
- Hugging Face Transformers
- Sentence Transformers

## Features

- Humanize AI text
- Preserve original meaning
- Casual Mode
- Professional Mode
- Academic Mode
- Creative Mode
- Meaning Preservation Score

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
